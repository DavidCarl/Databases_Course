## Usage

You can see below how to use my program.

### Save to DB file
```python main.py save -k <key_value> -v <value_here>```

### Read from DB file
```python main.py read -k <key_value>```

### Help Commands
 ```python main.py --help```

 ```python main.py save --help```
 
 ```python main.py read --help```
